ℹ️ Panduan Penggunaan Aplikasi
Langkah setup awal dan cara menjalankan PPAK BUMDesa
💡 Aplikasi ini dapat digunakan oleh BUM Desa maupun BUM Desa bersama yang memiliki berbagai kegiatan usaha jasa, dagang dan manufaktur sekaligus.
📋 Langkah Setup Awal
1. Isi identitas BUMDesa di halaman ini
2. Sesuaikan Daftar Akun (COA) dengan kebutuhan
3. Input Anggaran BUMDes di kolom anggaran
4. Isi nama petugas akuntansi dan pejabat penandatangan
✏️ Langkah Input Transaksi
1. Buka menu Jurnal Transaksi
2. Analisis transaksi dan pilih akun yang sesuai
3. Input jumlah Debit dan Kredit (double entry)
4. Isi tanggal, no. bukti, dan keterangan transaksi
⚠️ Persamaan Dasar Akuntansi: ASET + BEBAN = UTANG + MODAL + PENDAPATAN. Setiap transaksi harus seimbang (Debit = Kredit).
